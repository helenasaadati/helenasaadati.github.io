## 🛠️ Run Locally

Clone the repository:

```bash
git clone https://github.com/helenasaadati/helenasaadati.github.io.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

---

## 📦 Build

```bash
npm run build
```

---

## 🚀 Deploy

```bash
npm run deploy
```
